# DeskImage
  
<p align="center">
  <i align="center">Customizable frameless images pinned to your desktop<br>minimal, snappy, and always at hand.</i>
</p>

## 🚀 Quick Start

1. Clone the repo
2. dotnet build
3. Run DeskImage.exe
4. Drag an image into the frame

## ⌨️ Tips

- Right-click the title bar for options and customization.
